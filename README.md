
# Finance Tracker Frontend

Frontend for Finance Tracker built with:

- Next.js
- React
- Redux Toolkit (RTK Query)
- Tailwind CSS
- Chart Visualization

---

## Features

User Authentication  
 Dashboard Overview  
 Add Income / Expense  
 Charts & Analytics  
 Responsive Design

---

## 🏗 Tech Stack

- Next.js
- React
- Redux Toolkit
- RTK Query
- Tailwind CSS
- Recharts

---

## Installation

### 1. Clone Repo

```bash
git clone https://github.com/EphriamHab/finance-tracker-frontend.git
cd finance-tracker-frontend

```

### 2. Install Dependencies

```bash
npm install
```


### 3. Run Development Server

```bash
npm run dev
```

### 4. Live Deployment
The app is deployed and accessible at:  
👉 [https://finan-tracker.netlify.app/](https://finan-tracker.netlify.app/)  

To deploy your own version:  
1. Clone the repository  
2. Install dependencies with `npm install` or `pnpm install`  
3. Set up environment variables (`.env`) for database and Cloudinary  
4. Deploy to [Netlify](https://netlify.com) or your preferred platform
